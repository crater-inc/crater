## ビジュアルトーン（最重要）

このデザインが目指すのは**「エディトリアル・ドキュメンタリー」**。

- 写真週刊誌・ファッション誌のような誌面構成
- 写真が主役。テキストは写真を邪魔しない
- モノクロ・低彩度の写真が基本。色で語らず、構図と光で語る
- 「かっこいい」が先にあって、読み進めると「信頼できる」に変わる構造
- 装飾しない。削ぎ落とした先にある美しさ

## カラーパレット（厳守）

| 用途 | カラー | HEX |
|------|--------|-----|
| 背景（メイン） | 純白 | `#FFFFFF` |
| 背景（ダーク） | 純黒 | `#000000` |
| テキスト（メイン） | ほぼ黒 | `#0A0A0A` |
| テキスト（サブ） | ミディアムグレー | `#666666` |
| テキスト（ダーク面） | 白 | `#FFFFFF` |
| テキスト（ダーク面・サブ） | ダークグレー | `#444444` |
| 罫線（ライト面） | 薄いグレー | `#E0E0E0` |
| 罫線（ダーク面） | 暗いグレー | `#1A1A1A` |
| アクセント | 蛍光オレンジ | `#FF5C00` |

**アクセントカラーの使い方（重要）**
- 使いすぎない。ページ全体で5〜8箇所のみ
- セクションラベル・強調ワード・CTAボタンに限定
- それ以外は黒・白・グレーで構成する

## タイポグラフィ（厳守）

- **英字の大見出しは全大文字**（text-transform: uppercase）
- **見出しは大きすぎるくらいが正解**（clamp使用・画面幅の8〜10%）
- **中央揃えは絶対に使わない**（全て左揃え）
- **日本語本文**: font-size: 14px、line-height: 1.9
- **数字はMontserrat 900**: 価格・期間・番号を超大きく見せる
- **セクションラベル**: 英字10px・letter-spacing: 0.25em・オレンジ色

```css
/* 超大見出しの例 */
font-family: 'Montserrat', sans-serif;
font-size: clamp(56px, 10vw, 140px);
font-weight: 900;
letter-spacing: -0.04em;
text-transform: uppercase;
line-height: 0.95;
```

## レイアウト原則（厳守）

### 基本構造
- **白セクションと黒セクションを交互に配置**（白→黒→白→黒のリズム）
- **最大幅は設けない**。写真・背景は画面端まで伸ばす
- **コンテンツ余白**: padding 120px（上下）、48px（左右）
- **グリッドのgap**: 1〜2pxの細い隙間（セルの区切りがデザイン要素になる）

### 写真の扱い
- **複数枚グリッド**: 2〜4枚を隙間なく（1〜2px）並べる
- **縦横混在**: アスペクト比が違う写真を意図的に組み合わせる
- **フィルター**: `filter: grayscale(30%) brightness(0.85)` を基本とする
- **ダーク面の写真**: `filter: grayscale(40%) brightness(0.6)`
- **object-fit: cover** で統一。写真は絶対にトリミング優先

### 非対称レイアウト
- **左右均等を避ける**: 2カラムなら `55% / 45%` や `2fr / 1fr` など
- **大小のコントラスト**: 大きな写真1枚 + 小さな写真2枚の組み合わせ
- **テキストブロックは細く深く**: 幅を絞って縦に伸ばす

## UIスタイル

### セクションラベル（必須）
全セクションの冒頭に配置。英字・オレンジ・ライン付き。
```css
font-family: 'Montserrat', sans-serif;
font-size: 10px;
font-weight: 700;
letter-spacing: 0.25em;
color: #FF5C00;
text-transform: uppercase;
display: flex;
align-items: center;
gap: 14px;
/* ラベルの後に細線 */
::after { content:''; width:36px; height:1px; background:#FF5C00; }
```

### ボタン
- **CTAボタン**: 背景 `#FF5C00`、白文字、border-radius: 2px、height: 52px
- **アウトライン**: border: 1px solid currentColor、背景透明、border-radius: 2px
- **ダーク面のCTA**: オレンジ背景 or 白アウトライン

### 罫線・区切り
- 1px の細線のみ使用（太い罫線は使わない）
- グリッドのセル区切りは `gap: 2px; background: #E0E0E0`（ライト面）
- ダーク面は `gap: 2px; background: #1A1A1A`

## 各セクションのデザイン指示

### Hero（全画面）
- 黒背景・フルスクリーン（min-height: 100vh）
- 3〜4枚の写真を横グリッドで分割表示（絶対配置）
- 下からグラデーションオーバーレイ（rgba(0,0,0,0.8) → transparent）
- テキストは画面左下に配置（z-index: 1）
- 見出しは画面幅いっぱいの超大文字

### 課題セクション（黒背景）
- 左：写真1枚（縦長、aspect-ratio: 3/4）
- 右：テキストリスト（番号 + 強調テキスト + 説明文）
- 番号はアクセントカラー・Montserrat 700

### コンセプトセクション（白背景）
- 3カードを横並び（gap: 2px・グリッド区切り）
- 推奨オプション（顧問サービス）は黒背景カード

### サービスセクション（薄グレー背景）
- 写真左（55%）、テキストリスト右（45%）
- 下部にフルワイドの黒い帯（対応領域表示）

### メリットセクション（白背景）
- 3カラム・番号を超大きく（96px・Montserrat 900・薄グレー色）

### タイムラインセクション（黒背景）
- 上段：写真グリッド3枚
- 下段：テキスト3カラム（月数を超大きく）

### FAQセクション（薄グレー背景）
- 左カラム：ラベル＋見出し＋説明（固定幅260px）
- 右カラム：アコーディオンリスト

### 料金セクション（白背景）
- 2カラム（白カード / 黒カード）
- 価格数字を最大サイズで見せる（Montserrat 900）

### CTAセクション（黒背景）
- 左：背景写真＋テキスト
- 右：フォーム（黒背景）

## このデザインの「してはいけないこと」

- 中央揃えにしない
- ボーダーを太くしない（1px厳守）
- 背景をグラデーションカラーにしない（黒・白・薄グレーのみ）
- アイコンをアイコンフォントで入れない
- カードに大きなborder-radiusをつけない（2px以下）
- アクセントカラーを多用しない
- 写真をカラーのままにしない（必ずgrayscaleフィルターをかける）
