## 4. サービス概要

### セクションタイトル
```
どんなことをするのか
```

### 実務の流れ
```
【週1回 1時間のオンラインレビュー】

① 事前に制作データを共有
   ↓
② オンライン画面共有でレビュー
   アートディレクターが改善点を指摘、
   理由を説明し、一緒に修正を検討
   ↓
③ 必要に応じて録音データを共有
   後で何度も見直せる

【制作物がない場合】
レベルに合わせた課題を提示し、
実務スキルの向上をサポート
```

### 対応領域
```
グラフィック / Web / 資料 / パッケージ / 販促物 / 広告

「デザイン全般」に対応しています。
（コーディング除く）
```

### 月1回の育成レポート
```
毎月、経営者向けに
「社員の成長度」「改善すべき点」「次月の課題」
を報告書でお届けします。

進捗が可視化されるので、
投資効果を実感できます。
```

## 8. FAQ

### Q1: 週1回1時間でデザイン力は伸びますか？

```
A: はい、伸びます。

重要なのは「レビューの回数」ではなく「レビューの質」です。

実務制作物に対する具体的なレビューを受けることで、
社員は「なぜそのデザインが良いのか」を学びます。

この「理由を知る」という経験が、
デザイン力の根本的な成長につながります。

制作物がない場合も、
レベルに合わせた課題を提示し、
スキル向上をサポートします。
```

### Q2: 対応できるデザイン領域は？

```
A: グラフィック、Web、資料、パッケージ、販促物、広告など
「デザイン全般」に対応しています。

コーディングは対象外ですが、
Web制作における事前相談は可能です。
```

### Q3: チャットでの質問はできますか？

```
A: レビュー時間内での対応を基本としています。

ただし、緊急時やどうしても困った際は、
例外的に対応可能です。

常時チャットサポートをご希望の場合は、
別途ご相談ください。
```

### Q4: 同時に複数人対応できますか？

```
A: はい、最大3名まで対応可能です。

ただし、レビュー時間は増えないため、
お一人あたりの成長速度は遅くなります。

少人数での集中レビューをお勧めしています。
```

### Q5: 6ヶ月で成果が出なかった場合は？

```
A: 開始時に目標を設定します。

デザインは数値評価が難しいですが、
以下を成果指標としています：

✓ 制作物のクオリティ向上
✓ レビュー改善の反映度
✓ 社員の成長度

毎月の育成レポートで、
進捗が可視化されます。
```

### Q6: デザインはセンスでは？成長しないのでは？

```
A: デザインは「センス」ではなく「経験と知識」です。

正確で的確なレビューを受けることで、
確実に成長します。

実際に、私たちのクライアント企業では、
専門教育を受けていない社員が、
6ヶ月で大きく成長しています。
```

### Q7: 担当者が異動・退職した場合は？

```
A: 解約可能です。

契約条件に基づき、
残り期間を日割り計算し、
半額をご請求させていただきます。
```

### Q8: 誰が担当しますか？

```
A: クレーター代表のアートディレクターが対応します。

デザイン業界20年以上の実績を持ち、
大手企業の外部アートディレクターとして
活躍している実績があります。
```
