# スキル定義：ブランド診断

## スキル名
**ブランド診断** - 既存ブランドの現状を分析し、強み・ズレ・改善提案をまとめる

## 目的
既存ブランドのVI・コミュニケーション・競合との差異を診断し、
「今どういう状態か」を可視化する。
新規ブランド構築前の現状把握、リブランディングの起点として使う。

---

## 起動時の動作

```
/ブランド診断 [ブランド名 or URL or 素材パス]

例：/ブランド診断 crater.co.jp
例：/ブランド診断 output/brief/を読んで
例：/ブランド診断 クレーター
```

インプットをもとに以下を自動収集・分析する：
- WebサイトURL → アクセスしてスクレイピング
- 素材ファイル → 読み込んで分析
- ブランド名だけ → 推定で分析（仮説ベースと明示）

---

## 診断レポートの構成

```
【ブランド診断レポート：[ブランド名]】

■ 現状サマリー
  - ブランドが今どう見えているか（第三者視点で）

■ 強み（続けるべきもの）
  - VIの強み
  - コミュニケーションの強み
  - ポジショニングの強み

■ ズレ・課題
  - VIの課題（一貫性・時代感・差別化）
  - コミュニケーションの課題（誰に・何を・どう言うか）
  - ポジショニングの課題（競合との差異）

■ 競合比較
  - 主要競合3社との比較表

■ 改善提案（優先順位つき）
  1. すぐできること（クイックウィン）
  2. 中期でやること（VIリニューアルなど）
  3. 長期でやること（ブランド再定義）

■ 次のアクション
  → /ブランドマネージャー でリブランディング開始
  → /ブランドボイス でトーン再設計
  → /VI設計者 でVI刷新
```

---

## 分析観点

### VI（ビジュアルアイデンティティ）
- カラーの一貫性・時代感・差別化
- タイポグラフィの適切さ
- ロゴ・マークの強度
- デザインの統一感（Web・SNS・印刷物）

### コミュニケーション
- ブランドボイスの一貫性
- メッセージの明確さ
- ターゲットとのズレ
- SNS・Webのトーン統一

### ポジショニング
- 競合との差別化
- 独自性・記憶に残るか
- 価格帯・品質感との整合性

---

## 禁止事項

- ポジティブだけで終わらない（課題を正直に伝える）
- 抽象的な提案で終わらない（具体的なアクションを必ず書く）
- 情報が少なくても止まらない（仮説ベースで進め、仮説と明示する）
